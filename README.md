# Проектирование сервиса агрегации статистики

Этот репозиторий содержит документ с высокоуровневым дизайном для модуля «Менеджер статистики».

Финальная версия дизайна находится в файле [`DESIGN.md`](DESIGN.md).

Предложенная архитектура — это **модульный монолит**, построенный с использованием **Hexagonal архитектуры (порты и адаптеры)**. Это **событийно-ориентированная система**, использующая Kafka для надёжности и Redis для производительности. Она спроектирована так, чтобы быть надёжной, масштабируемой и быстро реализованной в соответствии с целями MVP.
